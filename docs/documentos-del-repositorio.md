# Documentos del repositorio

Este archivo lista los documentos de respaldo del repositorio Noti Inclusivos.

- `sustento-del-proyecto-personal.md`: documento central del proyecto personal.
- `autoria-y-contexto.md`: autoría, contexto de uso y reconocimiento.
- `alcance-comunicacional.md`: alcance educativo, informativo y comunicacional del boletín.
- `fuentes-y-creditos.md`: fuentes, créditos y recursos externos.
- `uso-permitido.md`: usos permitidos y condiciones generales.
- `criterios-de-publicacion.md`: criterios para seleccionar, revisar y publicar contenidos.
- `estructura-del-proyecto.md`: organización de archivos y carpetas.
- `bitacora-de-cambios.md`: registro de avances y mejoras.

Estos documentos ayudan a conservar el orden, la finalidad educativa, la autoría, la trazabilidad y el uso responsable del boletín digital.
